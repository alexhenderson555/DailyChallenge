# Мой челлендж на день — продуктовая спецификация

## Структура экранов

### Нижняя навигация (4 пункта)

| Вкладка       | Иконка (идея) | Экраны внутри |
|---------------|----------------|----------------|
| Главная       | Дом / звезда   | HomeScreen |
| Календарь     | Календарь      | CalendarScreen |
| Статистика    | График / тренд | StatsScreen (премиум: расширенная) |
| Настройки     | Шестерёнка     | SettingsScreen, подэкраны |

---

## 1. HomeScreen (Главная)

**Назначение:** показать цель на сегодня и дать отметить выполнение.

| Блок | Содержимое | Бесплатно / Премиум |
|------|------------|----------------------|
| Дата | «Пятница, 14 февраля» или «Сегодня» | — |
| Стрик | «🔥 5 дней подряд» (число + иконка) | Базовая счётка — бесплатно; «заморозка» стрика — премиум |
| Карточка цели | Текст цели, иконка категории, кнопки «Выполнил» / «Не выполнил» | — |
| Заменить цель | Кнопка «Другая цель» (показать другую из выбранных категорий) | Лимит раз в неделю — бесплатно; без лимита — премиум |
| Пустое состояние | Иллюстрация «Цель подгружается» / «Выбери категории в настройках» | — |

**Действия:** тап «Выполнил» → анимация успеха; «Не выполнил» → запись, без анимации праздника. Тап по карточке — опционально открыть подсказку/совет по цели (если добавишь контент).

---

## 2. CalendarScreen (Календарь)

**Назначение:** история — какие дни выполнены, какие нет.

| Блок | Содержимое | Бесплатно / Премиум |
|------|------------|----------------------|
| Выбор периода | Сегмент или табы: Неделя / Месяц | Неделя — бесплатно; Месяц — премиум (или оба бесплатно, премиум — экспорт) |
| Сетка календаря | Дни: иконка/цвет «выполнено» / «не выполнено» / «пропуск» / «нет данных» | — |
| День при тапе | Bottom sheet или экран дня: какая была цель, отметка, категория | — |
| Пустое состояние | «Здесь будет твоя история» + иллюстрация | — |

**Визуал:** одна ячейка = один день; цвет или иконка (галочка / крестик / минус). Единый стиль с темами.

---

## 3. StatsScreen (Статистика)

**Назначение:** прогресс и мотивация.

| Блок | Содержимое | Бесплатно / Премиум |
|------|------------|----------------------|
| Текущий стрик | Крупно: «12 дней подряд» | — |
| Лучший стрик | «Рекорд: 21 день» | — |
| За период | «За месяц: 18 из 28» (выполнено/всего) | Бесплатно: только месяц; премиум: неделя/месяц/год |
| График/визуал | Полоска или простой график выполнения по дням | Премиум: графики, «год в пикселях» |
| По категориям | «Чаще всего выполняешь: Здоровье» | Премиум |
| Экспорт | Кнопка «Сохранить отчёт PDF/CSV» | Премиум |

**Пустое состояние:** «Выполни первую цель — здесь появится статистика».

---

## 4. SettingsScreen (Настройки)

**Назначение:** время напоминания, категории, темы, премиум, прочее.

| Пункт | Действие | Бесплатно / Премиум |
|-------|----------|----------------------|
| Время напоминания | TimePicker — во сколько показывать цель дня | Одно время — бесплатно; второе напоминание — премиум |
| Категории целей | Список категорий, чекбоксы (какие показывать в ротации) | 1–2 категории — бесплатно; все + свои — премиум |
| Свои цели | Добавить свою цель (текст, категория) | Премиум |
| Тема оформления | Список тем с превью (Светлая, Тёмная, AMOLED, сезонные…) | 1–3 темы — бесплатно; все темы — премиум |
| Звук / вибрация | Вкл/выкл при отметке «Выполнил» | — |
| Виджеты | Короткая подсказка «Добавить виджет: long press на главном экране» | — |
| О приложении | Версия, ссылка на политику, контакты | — |
| Premium | Кнопка «Перейти на Premium» → экран оплаты/подписки | — |

---

## 5. Подэкраны и модалки

| Экраны | Описание |
|--------|----------|
| Онбординг | 2–4 экрана: что такое челлендж, выбор времени напоминания, выбор категорий. Кнопка «Начать». |
| Детали дня | Из календаря: дата, цель, отметка, категория. Только просмотр. |
| Выбор другой цели | Модалка или экран: показать 2–3 альтернативные цели из выбранных категорий, выбрать одну. |
| Premium / оплата | Описание преимуществ, кнопка подписки, восстановление покупок. |
| Добавить свою цель (премиум) | Поле текста, выбор категории, кнопка «Добавить». |

---

## 6. Виджеты (главный экран устройства)

| Виджет | Размер | Содержимое |
|--------|--------|------------|
| Цель дня | Малый / 2x2 | Текст цели (1 строка), кнопки «Выполнил» / «Нет». |
| Прогресс | Средний / 4x2 | Стрик, мини-календарь на неделю (точки/цвета). |
| Мотивация | Малый / 1x1 | Короткая фраза + иконка. |

Виджеты используют выбранную тему и шрифт приложения.

## Приоритет фич по версиям

### v1.0 (MVP) — бесплатно

- [ ] Онбординг: 2–3 экрана, выбор времени напоминания и 1 категории.
- [ ] Главная: цель на день, кнопки «Выполнил» / «Не выполнил», одна категория (или 2 на выбор).
- [ ] Логика смены цели раз в сутки в настраиваемое время.
- [ ] Календарь: вид «неделя», отметки выполнено/не выполнено.
- [ ] Статистика: текущий стрик, «за месяц» (выполнено/всего).
- [ ] Настройки: время напоминания, выбор 1–2 категорий, одна тема (светлая).
- [ ] Напоминание (push или локальное) в выбранное время.
- [ ] Пустые состояния с текстом (иллюстрации можно добавить в v1.1).

### v1.1 — полировка

- [ ] Мультяшный шрифт (Nunito / Comfortaa) везде.
- [ ] Тёмная тема + AMOLED (или одна тёмная).
- [ ] Анимация при «Выполнил» (галочка/конфетти).
- [ ] Иллюстрации: пустые экраны, успех, онбординг.
- [ ] Виджет «Цель дня» (малый).

### v2.0 — премиум

- [ ] Подписка / разовая покупка Premium.
- [ ] Все категории + свои цели (премиум).
- [ ] Замена цели без лимита (премиум).
- [ ] Второе напоминание, гибкое время (премиум).
- [ ] Календарь: месяц, экспорт (премиум).
- [ ] Статистика: графики, по категориям, экспорт PDF/CSV (премиум).
- [ ] Дополнительные темы (премиум).
- [ ] Виджет «Прогресс» (премиум или бесплатно — на твой выбор).
- [ ] «Заморозка» стрика 1 раз в N дней (премиум).

### Дальше (по желанию)

- [ ] Звуки при отметке (вкл/выкл в настройках).
- [ ] Виджет «Мотивация».
- [ ] Сезонные темы.
- [ ] Синхронизация / аккаунт (облако).
- [ ] Достижения и бейджи.

## Стек и данные (рекомендация)

- **Экраны:** Jetpack Compose (как в NearbyRoutes).
- **Хранение:** DataStore или Room — дата, цель дня, отметка (done/skip), выбранные категории, время напоминания, тема.
- **Цели:** локальный JSON или хардкод списков по категориям; рандом/выбор без повтора N дней.
- **Премиум:** Billing Library (подписка или разовая покупка), флаг `isPremium` в настройках.

Если нужно, следующим шагом могу набросать структуру папок и список экранов в коде (Compose) под Android.
