Актуальный список после последних доработок.

## Уже сделано (для справки)

- **Локализация:** все экраны (Home, Tabs, Calendar, Stats, Onboarding, Settings), категории, даты по локали приложения.
- **Темы:** светлая / тёмная / AMOLED бесплатно; тема «Как на устройстве» по умолчанию.
- **Календарь:** переключатель Неделя / Месяц, навигация по месяцам (стрелки), вид «Месяц» — только Premium.
- **Статистика:** год в пикселях, блок «По категориям», достижения (без экспорта).
- **Выбор цели:** бесплатно — 1 замена в день (случайная); Premium — 1 выбор в день из 4 вариантов (модалка).
- **О приложении:** версия, ссылки на политику и контакты (плейсхолдеры URL), кнопка «Восстановить покупки».
- **Онбординг:** тексты в строках, подсказка про замену/выбор цели, футер «категории/тема/язык в Настройках», иконка на первом экране.
- **Premium-описания:** benefit_3 — выбор из 4 целей; benefit_5 — календарь по месяцам.
- **Заморозка стрика:** Premium, 3 заморозки при активации, кнопка «Заморозить» после «Не выполнил», учёт в стрике.
- **Достижения:** по стрику (7, 30 дней), по целям (10, 25, 50, 100, 200, 500), по категориям (5 в каждой).
- **Анимация при «Выполнил»:** оверлей с галочкой (масштаб + затухание).
- **Иллюстрации:** иконки на пустых состояниях (главная — категории, статистика — нет данных), иконка в онбординге.
