# Мой челлендж на день

## Что делает продукт

Android-приложение для **одной цели в день**: напоминание в выбранное время, отметки «выполнил / не выполнил», календарь и статистика (стрик, рекорд, месяц). Подробная спецификация: **[docs/PRODUCT_SPEC.md](docs/PRODUCT_SPEC.md)**.

## Преимущества

- Минималистичный фокус — одна цель, без перегруза списками.
- Локальные данные через **DataStore**, напоминания через **AlarmManager** (в т.ч. после перезагрузки).
- Темы: светлая / тёмная / AMOLED.

## Установка

- Нужен **JDK 17** (в Android Studio обычно уже bundled).
- **Android Studio:** File → Open → папка `DailyChallenge` → дождаться Gradle → **Run**.
- **Сборка APK из консоли** (из корня проекта, JDK 17 в `PATH` / `JAVA_HOME`):
  - Windows: `gradlew.bat assembleDebug`
  - macOS/Linux: `./gradlew assembleDebug`  
  APK: `app/build/outputs/apk/debug/app-debug.apk`.

## Траблшутинг

| Проблема | Что проверить |
|----------|----------------|
| Gradle не синхронизируется | Интернет, прокси, версия JDK (должна быть 17). |
| Нет уведомлений | Разрешения приложения, оптимизация батареи OEM, время напоминания в настройках. |
| Данные «сбросились» | Редко — очистка данных приложения; бэпа нет, это MVP. |
| Краш на устройстве | Logcat в Android Studio, версия ОС и модель для issue. |

## Что сделано (v1.0 MVP)

- Онбординг: приветствие, время напоминания, 1–2 категории целей.
- Главная: цель на день, «Выполнил» / «Не выполнил», «Другая цель», стрик.
- Календарь: месяц с отметками.
- Статистика: серия, рекорд, месяц.
- Настройки: время, тема, звук, категории.
