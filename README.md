# Мой челлендж на день

Ежедневное приложение: одна цель в день в настраиваемое время, отметка выполнил/не выполнил, календарь и статистика.

## Документация

- **[Продуктовая спецификация](docs/PRODUCT_SPEC.md)** — структура экранов, фичи по версиям (MVP, полировка, премиум), виджеты, темы.

## Как собрать

- Нужен **JDK 17** (Android Studio обычно подставляет свой).
- В Android Studio: **File → Open** → папка `DailyChallenge` → дождаться синхронизации Gradle → **Run**.
- Или из терминала (из корня проекта, при установленном JDK 17):
  - Windows: `gradlew.bat assembleDebug`
  - macOS/Linux: `./gradlew assembleDebug`  
  APK: `app/build/outputs/apk/debug/app-debug.apk`.

## Что сделано (v1.0 MVP)

- Онбординг: приветствие, выбор времени напоминания, выбор 1–2 категорий целей.
- Главная: цель на день, кнопки «Выполнил» / «Не выполнил», кнопка «Другая цель», счётчик стрика.
- Календарь: месяц с отметками выполнено/не выполнено.
- Статистика: текущая серия, рекорд серии, за месяц.
- Настройки: время напоминания, тема (светлая/тёмная/AMOLED), звук при отметке, выбор категорий.
- Ежедневное напоминание в выбранное время (AlarmManager), перепланирование после перезагрузки.

## Дальше (v1.1+)

- Виджет «Цель дня» на главный экран.
- Мультяшный шрифт (Nunito): положить `.ttf` в `app/src/main/res/font/` и подключить в `Type.kt`.
- Анимация при «Выполнил», иллюстрации пустых экранов.

## Стек

- Android, Kotlin, Jetpack Compose, Material 3
- DataStore (настройки и история дней)
